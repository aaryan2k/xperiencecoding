Xperience Coding

We are a non-profit devoted to helping young kids who want to learn how to code

Projects

Xperience Coding helps kids learn coding through a 10-level course, where they learn the basics through game development. We also have camps!

Skills

Some of the skills that kids will learn is Object Oriented Programming (OOP), changing variables, using conditional statements, and more!

Network

Networking is a large part of our camps. Kids interact with other camp members to create ideas for projects and interact with helpers for assistance.
